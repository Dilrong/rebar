# Rebar (철근) 산업적/브루탈리즘 디자인 개편안

사용자의 피드백을 수용하여, 앱의 본래 이름인 **'Rebar(철근)'**가 가지는 단단하고 건축적인 의미에 맞게 디자인 시스템을 전면 재설계(Pivot)합니다. 기존의 부드럽고 따뜻한 독서 앱 스타일에서 벗어나, 데이터를 적재하고 제련하는 '공장'이나 '인프라'의 느낌을 주는 **Industrial & Brutalist (산업적 브루탈리즘)** 디자인으로 변경합니다.

## 🎨 1. 디자인 컨셉: "Raw Infrastructure (날것의 인프라)"
- **단단함과 견고함**: 모든 UI 요소는 건축물의 철근과 콘크리트처럼 투박하지만, 극도로 실용적이고 단단하게 보관됨을 시각화합니다.
- **가식 없는 UI**: 그림자(Shadow), 흐림(Blur), 둥근 모서리(Border-radius) 같은 장식적 요소를 모두 제거하고, 굵직한 직선과 명확한 구획을 사용합니다.

## 📐 2. 핵심 디자인 시스템 (Design System)

### 폰트 (Typography)
- **Monospace (포인트/메타데이터)**: 코딩 폰트인 `Space Mono`나 `Roboto Mono`를 사용하여 데이터의 고유값, 상태(INBOX, ACTIVE), 날짜 등을 터미널처럼 표현합니다.
- **Sans-serif (본문/제목)**: `Inter`를 가장 두꺼운(Black, ExtraBold) 형태로 제목에 사용하여 육중한 느낌을 주고, 본문은 가독성 있게 유지합니다. (Serif는 제거합니다.)

### 컬러 팔레트 (Color Palette)
"철근과 콘크리트, 그리고 현장"
- **Light Mode**: 콘크리트 그레이(`#F5F5F5` 부근) 배경에 칠흑 같은 완전한 검은색(`#111111`) 테두리와 글씨를 사용합니다.
- **Dark Mode**: 완전한 블랙(`#000000`) 배경에 밝은 회색/흰색 글씨를 씁니다.
- **Accent Color**: 현장의 경고/주목 색상인 **건설 오렌지 (Construction Orange, `#FF5A00`)** 또는 **네온 그린**을 강렬한 포인트나 버튼에만 제한적으로 사용합니다.

### 기하학 및 레이아웃 (Geometry)
- **Sharp Corners (직각)**: `border-radius: 0`을 전면 적용하여 모든 버튼과 카드가 날카로운 직사각형이 되게 합니다.
- **Heavy Borders (굵은 선)**: `border-2`, `border-black` 등을 활용해 구획선과 요소들을 뚜렷하고 강렬하게 분리합니다.
- **Solid Shadows**: 부드러운 그림자(Drop-shadow) 대신 우측 하단으로 쏠린 단색(검은색)의 Box-shadow를 사용해 종이가 겹친 듯한, 혹은 철판이 겹친 형태의 브루탈리즘 스타일 요소를 줍니다.

## ✨ 3. 화면별 UX 디자인 포인트

### A. Home (홈) & Nav (네비게이션)
- 'Rebar' 로고를 거대한 검은색 대문자로 투박하게 박아 넣고, 네비게이션은 터미널의 커맨드처럼 직관적이고 딱딱하게 떨어지도록 만듭니다.

### B. Capture (수집)
- 폼 요소를 서류 작성 철이나 재고 관리 시스템의 터미널처럼 굵은 테두리의 사각형 인풋으로 만듭니다.

### C. Daily Review (데일리 리뷰)
- 텍스트 자체는 `Inter`로 크게 보되, 버튼은 "승인(Acknowledge)", "재검토(Resurface)" 와 같이 시스템명령어처럼 투박하게 만들어 동작 자체가 확실한 결단처럼 느껴지게 합니다.

### D. Library (보관함)
- 예쁜 카드가 아닌, 마치 데이터베이스의 테이블이나 스프레드시트의 셀을 나열한 것처럼 굵은 테두리와 그리드를 사용하여 시각적인 무게감을 줍니다.
